# БизЛевел: Концепция рекламного лэндинга

---

## Общая информация

**Продукт:** Мобильное приложение для предпринимателей  
**Регион:** Казахстан  
**Цель лэндинга:** Скачивание приложения (App Store / Google Play)  
**Тон:** Уверенный, современный, без пафоса. Уважение к интеллекту пользователя.

---

## Главная идея

Сайт — это первый «уровень» для пользователя. Скролл страницы = прогресс. Прогресс-бар заполняется по мере просмотра — к финалу 100%. Это сразу демонстрирует механику продукта и создаёт вовлечение.

---

## Визуальный стиль

### Референсы
- Linear (премиальный минимализм, уверенный тон)
- WHOOP (данные как доказательство)
- Headspace (персонажи во «взрослой» теме)
- Duolingo (геймификация, уровни, прогресс)

### Цвета
| Роль | HEX |
|------|-----|
| Фон | #0A0A0B |
| Фон карточек | #141416 |
| Текст | #FFFFFF |
| Текст вторичный | #8A8A8E |
| Акцент (бренд) | #2563EB |
| Акцент (премиум) | #7C3AED |
| Успех / прогресс | #10B981 |
| Тёплый акцент | #F59E0B |

### Шрифт
Inter — все начертания. Заголовки Bold, текст Regular.

### Анимации
- Scroll-triggered появление блоков (ease-out, 0.4s)
- Прогресс-бар заполняется синхронно со скроллом
- Hover на интерактивных элементах (scale 1.02)
- Typing effect в демо-чате с AI-тренерами
- Confetti или pulse на 100% прогресса

---

## Механика прогресса

Тонкий прогресс-бар под хедером. Заполняется от 0% до 100% по мере скролла. Показывает: «ты уже идёшь, продолжай в приложении».

### Видео-превью (ТЗ)
- Хронометраж: 60-90 секунд

- Финал: "Это был один уровень. Впереди — система."
- Стиль: динамичный, с акцентами на ключевых моментах

- Клик на вариант ответа → появляется ответ тренера (typing effect)
- Под чатом: "Это демо. В приложении — полноценный диалог."
